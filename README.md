# SuperHint
IDA plugin that allows users to add comments to variable hints

# Installation
Copy `SuperHint.py` to IDA Pro plugins folder

>**Tested version** : IDA pro 8.4

# How to use
![gif](superhint.gif)
1. Press `Shift`+`A` to enable the feature
2. Click a variable or struct field in the pseudo-code and press `Shift`+`A` to add a hint comment.  
>the hints will be stored in the  database file*

# Features
- Supports Local and global variables structure fields and functions
